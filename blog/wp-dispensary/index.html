WP Dispensary

2016-01-18T14:41:21 · Robert

Within the last few months I have been developing this plugin from the ground up. I’ve ensured that when the product was released, there would be a great use for it, and a demand for more solutions like it.

With a market that is 29 states strong right now, and more states positioned to legalize medical marijuana as well as recreational marijuana over the next 5-10 years, I want to cater to this growing industry and provide the professional websites that dispensary owners deserve.

Professional dispensary websites.

The goal of the WP Dispensary plugin is to give a dispensary owner the ability to add and edit their menu of products with ease, directly from their WordPress dashboard.

Mission accomplished!

Introducing WP Dispensary

WP Dispensary was built with ease of use in mind, so the setup process of the plugin is as simple as adding and activating the plugin, adding your content and sitting back to bask in the glory of how easy it was to set up.

There’s also a lot of cool things baked right into the WP Dispensary menu plugin.

Easy to manage menu, complete control over menu item details, your content is exportable, and is 100% Open Source.

WP Dispensary is free to use with no license restrictions or expensive setup fees. It’s as simple as downloading the plugin, installing it on your WordPress powered website, and adding content.

Installation is a breeze

Set up WP Dispensary in a single click right from your dashboard. The WP Dispensary plugin is hosted in the official WordPress plugin repository.

Once you have the plugin installed, you’ll need an easy way to display your menu, and that’s where the CannaBiz WordPress theme comes into play.

I personally crafted a WordPress theme that is fully integrated with the WP Dispensary plugin, giving you a website that showcases your menu in the most beautiful way possible.

REST API Integration

One of the items on the development list for this plugin from the moment I released it in November of last year was REST API integration.

As of version 1.1, WP Dispensary comes fully integrated into the REST API, giving developers the ability to pull data from the custom post types and showcase them with Javascript, on the web or within an app.

Download

You can download the WP Dispensary WordPress plugin via the WordPress plugin directory.

You can also download it in your WordPress dashboard by going to Plugins - Add New and searching for WP Dispensary.
